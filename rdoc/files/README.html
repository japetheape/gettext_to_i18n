GettextToI18n

=======

This plugin will convert all of your existing Gettext calls( _(‘some translation’) ) to something like I18n::t ’(controller|view).id’. It inserts the translations into a new locale file "config/locales/standard.yml".

Namespace

===

By default your application will be divided into namespaces. For each namespace.

Example

=

To convert all your views, controllers and models to the new format.

rake gettext_to_i18n:transform
